* This is the example application for ZK "Getting Started" articles:
	- Get ZK Up and Running with MVC
	- Get ZK Up and Running with MVVM
* We keep this project structure in WTP style instead of Maven standard style
 because the article teaches users to use Dynamic Web Project not Maven.	
* This project bundles with ZK CE 7.0.0
* You should run it in a servlet 3.0 container, e.g. Tomcat 7.0 or Jetty 8.1 
* Release Procedure:
	1. In eclipse, Team\Export and zip the project 
	2. set file name as "getzkup-yyyymmdd.zip".
	3. publish
